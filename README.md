**Lenpaste** is a web service that allows you to share notes anonymously, an alternative to pastebin.com.
Java Script is not used.
There is its own API.

## Features
- No need to register
- IP addresses are not saved
- Not use Java Script
- Has its own API
- Open source and self-hosted
